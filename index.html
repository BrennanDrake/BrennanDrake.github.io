Monsters, Inc Echo Mount

In this project, I wanted to create an interesting mount for my amazon alexa dot. In doing so, I decided to create a replica Scream Canister from Monsters Inc and with the help of some microcontrollers I was able to sync light and sounds.

This project tested my abilities in circuit design and allowed me to grow in my coding skills as well. I prototyped my circuit in EasyEDA, physically using cardboard and solder, and then finally with a fabricated pcb from JLC Pcb. I utilized a raspbery pi zero w for the brains and an array of transistors to allow for staggered lighting of the LEDs in sync with an audio file I extracted from Monster's University.

  • Skills Used Most: CAD Design, 3D printing, 3D printed finishing, circuit prototyping and assembly, and multithreaded Python coding.
  • Tools and Software: CR-10 Mini 3D printer, OnShape CAD Software (Free to college and highschool students), EasyEDA circuit design, Spyder Python IDE, Adobe Audition, Soldering Iron, Sanding Tools, Bondo, and Spraypaint

RFID SLAM With Turtlebot3

Full Scope 10-25-2022

Arduino Wiring

This project is in progress with thesis to be completed in April of 2023

The goal of my research is to implement static unique landmarks in a simulataneous localization process through the use of off the shelf RFID (NFC) readers and tags. Specifically I am using PN5180 readers and plan to fuse lidar data and these landmarks into an existing SLAM package such as GMapping with a Turtlebot3. At the moment I have conducted literature search and physical construction of the robot, I am currently constructing my control circuits and am planning on implementing glass fast-blow fuses for caution. The scope of this project is for use in environments such as warehouses and stores where isles and shelves are repeating thus being prone to wrong data association. Not only will unique RFID tags aid in localization, but will prove to be useful for product tracking and planogram compliance as well.

  • Skills Used Most: Arduino coding, 3D Printing, CAD Design (Solidworks), GPIO integration, SLAM Algorithm integration with GMapping (Python), Soldering, Linux Programming and Troubleshooting
  • Tools and Software: Turtlebot3 by ROBOTIS, PN5180 NFC Readers, Arduino Uno, ROS Noetic, Soldering Iron, OnShape CAD Software, Logic Level Shifters, various fasteners, wire, and solder
  • Category: Academic
