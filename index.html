Monster Inc Echo Mount

In this project, I wanted to create an interesting mount for my amazon alexa dot. In doing so I decided to create a replica Scream Canister from Monsters Inc and with the help of some microcontrollers I was able to sync light and sounds.

This project tested my abilities in circuit design and allowed me to grow in my coding skills as well. I prototyped my circuit in EasyEDA, physically using cardboard and solder, and then finally with a fabricated pcb from JLC Pcb. I utilized a raspbery pi zero w for the brains and an array of transistors to allow for staggered lighting of the LEDs in sync with an audio file I extracted from Monster's University.

  • Skills Used Most: CAD Design, 3D printing, 3D printed finishing, circuit prototyping and assembly, and multithreaded Python coding.
  • Tools and Software: CR-10 Mini 3D printer, OnShape CAD Software (Free to college and highschool students), EasyEDA circuit design, Spyder Python IDE, Adobe Audition, Soldering Iron, Sanding Tools, Bondo, and Spraypaint
